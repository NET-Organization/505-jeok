## 📝 오늘 공부한 내용
> AWS ML 서비스

AWS SSA 덤프를 풀고 있는데, 가끔씩 ML 문제가 나와서 정리해봤습니다.

### 1. Amazon Polly : TTS

- 텍스트를 음성으로 변환(TTS)

### 2. Amazon Transcribe : STT

- 음성을 텍스트로 변환(STT)

### 3. Amazon Sagemaker : 모델 구축&배포

- ML 모델 구축, 배포할 수 있는 완전관리형 서비스

### 4. Amazon Lex : 챗봇

- 챗봇이나 음성 인식 애플리케이션을 구축할 수 있는 대화형 AI 서비스

### 5. Amazon Rekognition : 부적절한 콘텐츠 탐지

- 이미지 및 비디오 분석 자동화
- 부적절한 콘텐츠를 탐지할 수 있음
- 보안, 모니터링에 쓸 수 있음

**Amazon Comprehend Medical**: 의료 관련 문서에서 PHI(Protected Health Information)와 같은 민감한 정보를 분석하고 추출하는 데 사용됨

### 6. Amazon Comprehend : nlp

- NLP 서비스
- 감성 분석, 키워드 추출, 주제 분석

### 7. Amazon Translate : 번역

- 번역 서비스

### 8. Amazon Textract : 텍스트 추출

- 문서에서 텍스트, 표, 양식 등의 데이터를 자동으로 추출할 수 있는 서비스

### 9. Amazon Personalize : 추천시스템

- 사용자의 행동 데이터를 분석하여 개인화된 추천을 제공하는 서비스

### 10. A**WS Deep Learning AMIs**

- AWS에서 제공하는 딥러닝용 Amazon Machine Image(AMI)
- AMI를 사용해 사전 구성된 환경에서 TensorFlow, PyTorch, MXNet 등의 프레임워크를 활용하여 딥러닝 모델을 훈련시킬 수 있음

### 11. AWS DeepLens : 실시간 영상 인식

- 실시간으로 영상 인식 기능 수행하는 **카메라 장치**
- 영상 처리 AI 모델을 빠르게 테스트하고 배포하는 데 유용

### 12. Amazon Kendra : 검색

- 검색 서비스
- 자연어로 질문하면 문서나 데이터베이스에서 관련된 정보를 찾아줌

### **13. Amazon Forecast : 시계열**

- **시계열** 예측 서비스
- 과거 데이터를 기반으로 미래의 수요나 매출 등을 예측하는 데 사용됨
    
    ex) 기후 데이터, 주가 지수 
    

### 14. **Amazon Fraud Detector : 온라인 사기 탐지**

- **온라인 사기 행위**를 탐지하고 예방하는 데 도움을 주는 서비스
- 사용자 계정 도용, 결제 사기 등을 방지하는 데 사용됨
