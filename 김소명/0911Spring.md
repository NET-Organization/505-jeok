## 😲 오늘 깨달은 내용
Java 문법에 Optional 이라는 기능을 알게 되었다.
함수가 반환될 때 Null 값이 오면 오류가 날 수 있으니까 Optional 아류 기능들을 사용해서 이를 방지한다.
